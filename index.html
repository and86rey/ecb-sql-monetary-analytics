How it works?

This application fetches real-time data from the European Central Bank (SDMX API) directly to your browser. The data is then parsed and loaded into an in-memory DuckDB database instance running in a Web Worker.

No data ever leaves your machine. This is a pure "Serverless" SQL architecture.

Data Source: ECB Statistical Data Warehouse. Powered by DuckDB-WASM.
