Data notes (what’s loaded?)

The app fetches ECB series via the ECB Data API and inserts observations into obs (long format: period, series_code, value). A small series table contains labels and keys.

Try: SELECT * FROM series; or SELECT * FROM obs LIMIT 10;

Runs fully in browser. If ECB is unreachable, you’ll see a clear error message and nothing is stored server‑side.
